# API 한도 · 반영시점 정리

각 데이터 소스의 일일 한도, 데이터 반영 시점, 비고를 정리한 참조 문서.

---

## 1. KRX OpenAPI (공식, `openapi.krx.co.kr`)

| 항목 | 내용 |
|------|------|
| 인증 | 인증키(`KRX_API`) |
| 일일 한도 | 공식 명시 없음 (실운영 기준 ~10,000건 가이드라인) |
| 반영 시점 | **전일 데이터**, 익일 영업일 오전 8시 업데이트 |
| 당일/실시간 | **불가** (일별 단위 데이터만 제공) |
| 주의 | pykrx 등 비공식 크롤링은 IP 차단 대상 (차단 해제 원칙적 불가) |

**사용 엔드포인트**: `stk_bydd_trd` (일별 매매정보), `stk_isu_base_info` (종목기본정보), `etf_bydd_trd` (ETF)

**결론**: 오전 8시 이후 1회 조회 → 하루 중 데이터 변동 없음.

---

## 2. DART OpenAPI (공식, `opendart.fss.or.kr`)

| 항목 | 내용 |
|------|------|
| 인증 | 인증키(`DART_API`) |
| 일일 한도 (개인) | **20,000건** (83종 전체 서비스 합산 기준) |
| 일일 한도 (기업\*) | 공시검색·기업개황 2종 **한도 없음**, 나머지 81종 20,000건 |
| 분당 한도 | 1,000회 이상 과도한 접속 시 제한 |

\* 기업: 사업자등록증 + IP 등록 필요

**반영 시점 (서비스별)**

| 데이터 | 반영 시점 |
|--------|----------|
| 고유번호(corp_code) 변경 | 익영업일 오전 |
| 공시원본·사업보고서 재무정보 | 공시 제출 후 **15분** (마감일 등 지연 시 익영업일 내 완료) |
| 재무정보 일괄다운로드 | 정기보고서 제출 마감 후 **7영업일 이내** |

**데이터 제공 범위**: 사업보고서 재무정보 2015년~

**사용 엔드포인트**: `fnlttSinglAcntAll` (재무제표), `company` (기업개황), `corpCode.xml` (corp_map)

**결론**: 사업보고서는 연 1회 제출 → 오전 8시 이후 1회 조회로 하루 중 변동 없음. `corp_map.json`, `sector_map.json` 캐시로 재호출 최소화.

## 3. KRX 로그인 (비공식, `data.krx.co.kr` 내부 API)

| 항목 | 내용 |
|------|------|
| 인증 | KRX 회원 로그인 (`KRX_ID`, `KRX_PW`) |
| 공식 한도 | **없음** (비공식 스크래핑) |
| 계정 잠금 위험 | 반복 로그인 실패·과도한 호출 시 잠금 |
| 반영 시점 | KRX OpenAPI와 동일 (전일 기준, 오전 8시~) |

**사용 엔드포인트**: `MDCSTAT03501` (PER/PBR/배당), `MDCSTAT00601` (코스피200 구성종목)

**무로그인 불가 여부**: 2026-06 현재 익명 POST → `LOGOUT`(HTTP 400) 차단 확인. 상세는 [DESIGN.md](DESIGN.md) 참조.

**코스피200 구성종목 갱신**: 연 2회(6·12월 정기변경) 시에만 실질적 변경.

**결론**: 일 1회 호출이면 계정 잠금 위험 낮음. PER/PBR은 공식 OpenAPI 미제공이라 이 경로가 유일한 실측값.

## 4. Naver OHLCV (비공식, `fchart.stock.naver.com`)

| 항목 | 내용 |
|------|------|
| 인증 | 없음 (User-Agent만) |
| 공식 한도 | **없음** (비공식 스크래핑) |
| 반영 시점 | 일봉 기준 전일 종가, **장 중에도 당일 데이터 업데이트** |
| 당일/실시간 | 일봉 단위로 조회 가능 (실시간은 아님) |

**사용**: 2단계 pool(상위 ~25종목) 기술적 지표 계산용 (RSI, 이동평균)

**결론**: 하루 1회 기준 25건 수준. 장 중 재조회가 필요하다면 Naver만 추가 실행해도 KRX/DART 한도 영향 없음.

## 요약표

| 소스 | 성격 | 일일 한도 | 오전 8시 이후 고정? |
|------|------|-----------|---------------------|
| KRX OpenAPI | 공식 | ~10,000건 | ✅ 하루 고정 |
| DART API | 공식 | 20,000건 | ✅ 하루 고정 (재무) |
| KRX 로그인 | 비공식 | 없음 | ✅ 하루 고정 |
| Naver OHLCV | 비공식 | 없음 | ⚠️ 장 중 변동 가능 |

**현재 스크리너 1회 실행 시 API 호출 수**: KRX 2~3건 + DART 50~80건 + Naver 25건 → 모두 한도 여유 있음.
